# 收付银行对照账号选择器自定义样式 操作记录

## 需求背景
在收付银行对照功能中，不仅需要在“银行账户号码”选择器的选项里显示账户名称，还需要将其做成左右两列对齐的特定样式（左侧显示深色账号，右侧显示浅色、字号较小的账户名称）。

## 分析思路
1. **分析 Element UI 下拉选项自定义方法**：
   - `<el-option>` 组件默认只显示 `:label` 绑定的值，但可以通过插入自定义的 HTML 模板来自定义下拉列表里的单行显示内容。
2. **样式布局设计**：
   - 账号在左侧：使用 `float: left` 靠左。
   - 账户名称在右侧：使用 `float: right` 靠右，并设置字体颜色为浅灰色（如 `#8492a6`），字号调小（如 `13px`）以弱化视觉效果，突出账号主体。
3. **兼容性**：
   - `:label` 依然保留为 `yhzhhm（yhzhmc）` 形式，以确保选中后回显在输入框里的值依然包含名称。

## 解决步骤
在 `src/components/cwhs/sfyhdz.vue` 文件中：
针对单选和多选的 `<el-option>`，内部加入 `<span>` 标签对：
```vue
<el-option v-for="item in yhzhxxs"
           :key="item.id"
           :label="`${item.yhzhhm}（${item.yhzhmc}）`"
           :value="item.id">
    <span style="float: left">{{ item.yhzhhm }}</span>
    <span style="float: right; color: #8492a6; font-size: 13px">{{ item.yhzhmc }}</span>
</el-option>
```

## 验证与留痕
- 通过本次修改，银行专户选择器的下拉选项成功实现了左侧深色大字账号、右侧浅色小字名称的优雅布局，选中后在输入框内依然显示完整信息。
